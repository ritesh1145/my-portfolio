# Personal Portfolio — Ritesh Singh

This is a modern, responsive personal portfolio built with HTML, CSS and Vanilla JavaScript.

Features:
- Dark theme with glassmorphism
- Smooth animations and scroll reveal
- Mobile-first responsive layout
- Typing effect, sticky navbar, active section highlighting
- Back-to-top button, project cards, contact form (mailto fallback)

To run locally:
1. Open `index.html` in your browser.

Live demo:
- Deployed: https://my-portfolio-psi-five-55.vercel.app/ (Ritesh Singh — Full Stack Developer)

Project structure:
```
portfolio/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   └── resume/
└── README.md
```

Notes:
- Replace placeholder images in `assets/images` and the resume in `assets/resume`.
- No build step required. Works with any static file server.
